# 🎨 Image to ASCII Converter

Image to ASCII Converter is a Python-based utility that transforms images into ASCII art using text characters.

The program allows users to select one or multiple images through a file browser, automatically converts them into grayscale ASCII art, and saves the results as numbered text files.

Perfect for learning image processing, ASCII art generation, and Python automation.

---

# ✨ Features

🖼 Convert Images to ASCII Art

* Transform images into text-based artwork

📂 Multiple Image Selection

* Process multiple images at once

🎨 Grayscale Conversion

* Converts images to grayscale before processing

📏 Automatic Image Resizing

* Maintains aspect ratio for better ASCII output

💾 Auto-Save Results

* Automatically generates numbered text files

📁 Organized Storage

* Saves all ASCII outputs in a dedicated folder

⚡ Fast Processing

* Lightweight and efficient conversion

---

# 🧠 How It Works

🖼 Image Loading

Loads selected images using Pillow.

📏 Resize Processing

Adjusts image dimensions while maintaining aspect ratio.

⚫ Grayscale Conversion

Converts images into grayscale values.

🔤 ASCII Mapping

Maps pixel brightness values to ASCII characters.

💾 Output Generation

Creates ASCII art and saves it as a text file.
